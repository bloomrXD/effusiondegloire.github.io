Pasteur principal — Église de la Grâce

Ce site vous permet de suivre, relire et réviser les messages dominicaux même si vous n'étiez pas présent. Le quiz de compréhension vous aide à ancrer la Parole dans votre cœur. Partagez cette page avec vos frères et sœurs !

Guide — Comment mettre à jour ce site
1
Ouvrez le fichier culte.html avec un éditeur de texte (Notepad, TextEdit ou VS Code).
2
Trouvez le bloc const CULTE_DATA = { tout en haut — c'est la seule zone à modifier. Changez le titre, la date, les sections, versets et questions du quiz.
3
Sauvegardez le fichier, puis allez sur votre dépôt GitHub. Cliquez sur le fichier culte.html, puis sur le crayon (✏️ Edit), collez le nouveau contenu et cliquez Commit changes.
4
Votre site se met à jour automatiquement en 1 à 2 minutes. L'URL reste la même — envoyez-la aux fidèles par WhatsApp ou email !
Héberger sur GitHub Pages

Étape 1 : Créez un compte sur github.com avec l'email de l'église.

Étape 2 : Créez un nouveau dépôt (repository) public. Nommez-le exactement : votre-pseudo.github.io

Étape 3 : Uploadez le fichier culte.html en le renommant index.html

Étape 4 : Allez dans Settings → Pages → Source : main branch → Save.

Résultat : Votre site est en ligne à l'adresse votre-pseudo.github.io — gratuitement et sans limite !
